> **개발 기간:** 2025.10 ~ 현재 (진행 중)  
> **서비스 소개:** 앨범 단위 리뷰, 커뮤니티 게시판, 오늘의 앨범 등 음악 감상 기록과 리스너 간의 취향 교감을 제공하는 웹 서비스

| 구분 | 정보 |
| --- | --- |
| **인원 구성** | 총 3명 (**본인 참여도: DB 설계 30%, 개발 50%**) |
| **Tech Stack** | `TypeScript`, `JavaScript`, `Next.js (App Router)`, `Vue.js`, `Supabase`, `PostgreSQL`, `Vercel` |
| **Tools & Design**| `Windows`, `Git`, `Notion`, `Figma` |

#### ✨ 주요 기능
- **음악 검색 및 홈:** 아티스트 검색(자동완성), 슬라이드바(추천/선정 앨범 마퀴 카드), 오늘의 앨범 탭 운영
- **깊이 있는 평론:** 싱글 위주가 아닌 0~10점 평점 기반의 '앨범 단위 리뷰' 작성 및 실시간 평균 평점 집계
- **소통 커뮤니티:** 국내/해외 음악 담론 게시판, 창작자 피드백용 워크룸, LP/CD 등 음악 물품 장터 운영
- **사용자 관리:** NextAuth 활용 이메일/Google 로그인, 프로필·리뷰 기록·즐겨찾기 관리 마이페이지 제공
- **어드민 어플리케이션:** 리뷰 승인 워크플로, 오늘의 앨범/슬라이드바/공지사항 편집 시스템 구현

#### 👨‍💻 나의 담당 업무
- **풀스택 핵심 개발:** Next.js·Supabase 기반의 음악 탐색 백엔드 기능 및 리뷰 CRUD 전담 구현
- **데이터 집계 및 연동:** 유저 평점 실시간 집계 연산 처리, 타인 피드 프로필 탐색 로직 구축
- **소셜 로그인 및 인프라:** NextAuth.js 연동 환경 설계 및 Vercel(icn1 리전)을 통한 배포 및 운영
