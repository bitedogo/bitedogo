## 📝 Detailed Project Specifications (상세 기술서)

### 💿 1. 음악 리뷰 사이트 'ORU'
> **기간:** 2025.10 ~ 현재 (진행 중)  
> **소개:** 음악 감상 기록과 리스너 간의 깊이 있는 취향 교감을 제공하는 풀스택 웹 서비스

*   **팀 구성:** 총 3명 (**본인 기여도: DB 설계 30%, 개발 50%**)
*   **Tech Stack:** `TypeScript`, `JavaScript`, `Next.js`, `Vue.js`, `Supabase`, `PostgreSQL`, `Vercel`
*   **Environment & Tools:** `Windows`, `Git`, `Notion`, `Figma`
*   **주요 기능 및 담당 업무:**
    *   **풀스택 기능 개발:** Next.js와 Supabase 환경을 아우르며 음악 검색 파이프라인 및 핵심 리뷰 CRUD API 전담 설계.
    *   **실시간 데이터 집계:** 유저들의 개별 별점 데이터를 실시간으로 집계 연산하여 신뢰도 높은 '평균 평점' 메트릭 및 타 리스너 피드 프로필 연동.
    *   **인증 및 배포 인프라 구축:** NextAuth.js 기반 Oauth2 구글 소셜 로그인 연동 및 Vercel 파이프라인 지속적 배포 완료.
*   **💥 Troubleshooting:**
    *   *Challenge:* 이종 음원 데이터 API 결합 시의 속도 병목 및 타깃 포맷 충돌로 프론트엔드 타입 안정성 저하.
    *   *Solution:* 비동기 병렬 처리 구조 최적화(`Promise.all`)로 레이턴시를 제어하고 포맷 파싱/매핑 레이어를 단일화하여 안정적인 유니온 타입 분기 렌더링 달성.

---

### 🐱 2. 온라인 자바 학습 플랫폼 '자바냥 (JavaNyang)'
> **기간:** 2025.01 ~ 2025.09  
> **소개:** 메가스터디교육 더조은컴퓨터아카데미 연계 프로젝트로 개발된 퀴즈 기반 인터랙티브 학습 플랫폼

*   **팀 구성:** 총 6명 (**본인 기여도: DB 설계 40%, 개발 20% / Team Leader**)
*   **Tech Stack:** `JavaScript`, `Next.js`, `Vue.js`, `Bootstrap`, `Supabase`, `PostgreSQL`
*   **Environment & Tools:** `Windows`, `Git`, `Figma`
*   **주요 기능 및 담당 업무:**
    *   **데이터 모델 관계 최적화:** 다대다 관계 테이블 구조를 정규화하고 운영 편의성을 위한 관리자용 회원 검색 및 상세 제어 시스템 전반을 구축.
    *   **경험치 및 업적 게임화(Gamification) 시스템:** 문제 난이도 계수를 산정하여 자동 경험치 부여 로직 및 유저 업적 연동 엔진 설계.
    *   **실시간 양방향 커뮤니케이션:** 회원가입 처리와 유기적으로 바인딩되는 실시간 대전용 소켓 채팅 핸들러 가발.
*   **💥 Troubleshooting:**
    *   *Challenge:* 다수 사용자가 퀴즈 세션에 동시 진입 시 데이터 커넥션 스파이크로 인한 동기화 지연 리스크.
    *   *Solution:* Supabase 실시간 상태 구독 범위 및 업데이트 빈도를 도메인 단위로 스로틀링(Throttling)하여 동기화 정밀도 극대화.

---

### 📚 3. 스터디 협업 플랫폼 'StudyO (스터디오)'
> **기간:** 2025.01 ~ 2025.09  
> **소개:** 메가스터디교육 더조은컴퓨터아카데미 연계 프로젝트로 개발된 통합 팀 빌딩 및 일정 관리 플랫폼

*   **팀 구성:** 총 6명 (**본인 기여도: DB 설계 30%, 개발 20%**)
*   **Tech Stack:** `Java`, `JavaScript`, `Next.js`, `Vite`, `Bootstrap`, `OracleDB`, `AWS`
*   **Environment & Tools:** `Windows`, `Git`, `Figma`
*   **주요 기능 및 담당 업무:**
    *   **마이페이지 대시보드 아키텍처:** 유저의 스터디 활성도 추적 및 주간 일정을 시각화하여 확인하는 복합 대시보드 패널 개발.
    *   **프로필 상태 비동기 관리:** 닉네임 변경 및 멀티파트 이미지 업로드 등 유저 코어 세션 정보 실시간 업데이트 기능 구축.
    *   **공용 컴포넌트 표준화:** 디자인 시스템에 대응하는 재사용 가능 모달(Modal) 추상화 컴포넌트 설계로 프로젝트 내 중복 코드 라인을 제거.
*   **💥 Troubleshooting:**
    *   *Challenge:* 캘린더 내 다용량 스케줄 조인 쿼리 호출 시 병목 연산으로 인한 프론트엔드 UI 스크롤 프레임 드랍 현상.
    *   *Solution:* 계층 간 데이터 이동 시 필수 필드만 직렬화하도록 DTO 구조를 리팩토링하고, React 렌더링 트리거 바인딩을 제거하여 렌더링 속도 개선.
